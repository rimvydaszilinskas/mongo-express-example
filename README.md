# Express.js + Mongo-Express + Mongo

```
docker-compose up --build
```

## Navigating

```
:3000 - Express app
:3001 - Mongo-Express
```

```
GET / - will return a 200 OK if up and runnign
GET /cities - returns a list of cities
POST /cities - create a city entry
GET /cities/:id - return a specific city
```
